### STEP 1 — claude-mem [INSTALL FIRST]

**Why first:** Starts capturing context immediately. Every action after this is persisted.

**Install (in Claude Code):**
```
/plugin marketplace add thedotmack/claude-mem
/plugin install claude-mem
```
Then restart Claude Code.

**What activates automatically:**
- `SessionStart` hook: injects context from last 10 sessions
- `PostToolUse` hook: captures observations from every tool call
- `Stop` hook: summarises session for future retrieval
- Web viewer: http://127.0.0.1:37777

**7 MCP tools unlocked:**
- `search` — natural language query over all project history
- `timeline` — chronological context around any result
- `get_observations` — fetch full details by ID
- `smart_search` — AST-based TypeScript symbol search (tree-sitter)
- `smart_unfold` — expand a symbol to full source
- `smart_outline` — file structural outline
- `__IMPORTANT` — workflow guidance

**PATS-Copy benefit:** Never re-explain the codebase architecture. Claude finds how `closePosition()` works, what `RANK_MULTIPLIERS` is, how `confirmWatcher()` uses AI thresholds — from memory, not from re-reading files.

**Conflict check:** NONE

### STEP 2 — Ruflo MCP [INSTALL SECOND]

**Why second:** Cost routing cuts AI spend ~60-70% immediately.

**Install MCP server:**
```bash
claude mcp add ruflo -- npx -y ruflo@latest mcp start
```

**Verify:**
```bash
claude mcp list
```

**Init vector memory (for trade pattern storage):**
```bash
npx ruflo@latest memory init
```

**Enable cost-based routing:**
```bash
npx ruflo@latest config set --provider-strategy cost-based
```

**3-tier routing for PATS-Copy:**
| Market Type | Tier | Model | Saving |
|-------------|------|-------|--------|
| Simple sports match result | 1 | WASM booster (<1ms) | ~95% |
| Crypto/politics (known pattern) | 2 | Haiku | ~90% |
| Novel/complex markets | 2 | Sonnet (current) | 0% |
| Ambiguous edge cases | 3 | Opus | -cost (only when needed) |

**Vector memory use case:**
Store closed trade outcomes as vectors → `search` for similar past markets → 4th corroboration signal in `ConfirmationLayer`:
```bash
npx ruflo@latest memory store --key "sports-correct-veto" --value "Chelsea PSG 0.82 vetoed correctly"
npx ruflo@latest memory search --query "football match result high price"
```

**Swarm topology (Phase 4 — UI Build, defer for now):**
```bash
npx ruflo@latest swarm init --topology hierarchical --agents 5
npx ruflo@latest swarm start
```
Assign: Agent 1 = Supabase API, Agent 2 = Next.js scaffold, Agent 3 = Charts, Agent 4 = Risk UI, Agent 5 = Leaderboard view

**40+ MCP tools include:** `swarm_init`, `agent_spawn`, `task_orchestrate`, `memory_usage`, `neural_train`, plus 35 coordination tools

**Conflict check:** NONE. MCP server runs as separate process. Bot TypeScript unchanged.

### STEP 3 — everything-claude-code [INSTALL THIRD]

**Why:** PM2 daemon management, security scanning, memory learning, 52+ slash commands.

**Install (Claude Code):**
```
/plugin marketplace add affaan-m/everything-claude-code
/plugin install everything-claude-code@everything-claude-code
```

**Install TypeScript rules (manual step):**
```bash
cd /tmp/POLYMARKET-3.0
npx everything-claude-code@latest install --target typescript
```

**Key commands unlocked for PATS-Copy:**
| Command | Use |
|---------|-----|
| `/pm2` | Start/stop/restart/log the copy-trading daemon |
| `/security-scan` | Audit private key + API credential handling in src/ |
| `/tdd` | Enforce correctness on `copy-executor.ts` logic |
| `/checkpoint` | Save state before risky changes to execution layer |
| `/model-route` | Manual cost routing when Ruflo not triggered |
| `/multi-backend` | Parallel workstreams (signals + execution + monitoring) |
| `/learn` | Auto-extract patterns from current session |
| `/evolve` | Update learned skills from session history |

**MCP count guard:**
After installing ECC (14 MCP servers) + Ruflo (40+ tools) + claude-mem (7 tools):
```bash
claude mcp list
```
Disable any ECC MCPs not relevant to a TypeScript trading bot:
```json
// .claude/settings.json
{
  "disabledMcpServers": ["ecc-python-server", "ecc-go-server", "ecc-swift-server"]
}
```
Target: keep active MCP tools ≤ 80.

**Conflict check:** NONE — purely dev layer, no app code touched.
